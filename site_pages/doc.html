Réglages de la portée

  • BEGIN line pour démarrer une portée / créer une nouvelle ligne
  • SET clef_sharp N pour indiquer qu'il y a N (1, 2, 3 etc.) dièses à la clef
  • SET clef_flat N pour indiquer qu'il y a N (1, 2, 3 etc.) bémols à la clef
  • SIGNATURE A B et SIGNATURE C : pour indiquer le rythme (exemple 4 4, 3 4, etc)

Ajouter des notes

La logique est la suivante : on indique le nom/numéro de la note ainsi que des options. Exemples :

  • do pour le do de base, équivalent à 1
  • re pour le ré, équivalent à 2
  • do+ pour le prochain octave, équivalent à 8

Concernant le type de note qui est utilisé (blanche, noire, croche etc.), il se change avec l'instruction SET duration N, avec N=0 pour une ronde, 1 pour une blanche, 2 pour une noire, etc.

Altérations

Elles se placent avant la note :

  • # dièse
  • b bémol
  • n bécart (n comme naturel)

Options

À placer après la note :

  • ! pour que la tige regarde vers le bas
  • . pour une note pointée

Piste de voix

Après les options, on peut faire correspondre la note avec des paroles en les indiquant [entre crochets], par exemple :

  • do[bon] re[-jour] Va écrire Bon jour sous les notes avec le tiret placé bien au milieu

Accords et triolets

On indique un accord en groupant les notes entre parenthèses, et on lie des croches grâce à des crochets

  • (do mi sol) pour l'accord de Do majeur
  • [re mi] [mi fa] [re fa] pour trois groupes de deux croches connectées

Les accords et les groupes peuvent pointer vers le bas en mettant un ! après les parenthèses/crochets.

Commandes avancées

  • MOVE X forward ou MOVE X back pour rajouter de l'espace (ou revenir en arrière, utile par exemple pour séparer plusieurs voix). X est la taille de l'espace ; X=1 correspond à l'espace après une noire ; X=0.33 correspond à la taille prise par un # ou bémol à la clé
